# Introduction to Git

## Create a new Git Repo


git init

## Check status of your repo
git status

## To add files to git
git+ filename1 filename2

## To commit a file
git commit -m'Message'

## Check git remote
git remote show origin

## git log / git branch
check branch

## push local branch to remote
 git push <remote name><branch name>

'git push origin main'
